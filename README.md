# spijker-tex
ASO Spijker LaTeX templates

## Voorbeeld
```
\usepackage[wiskunde, test]{spijkerarticle}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%\vak{Wiskunde}
\module{Test}
\titel{Thema 1}
\leerkracht{P. Smets}
%\datum{}
%\klas{}
\cijfer{20}
%\voornaam{}
%\naam{}
%\baseline{barst van wiskunde}
%\toelichting{eigen uitleg over de test}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{document}

\maketitle

\end{document}
```
